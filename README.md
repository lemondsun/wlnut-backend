## Setup
To run this project, install it locally using npm and run :

node index.js

Use the package manager [npm](https://www.npmjs.com/).

```bash
npm i --save axios
npm install -g --save-dev jest supertestnpm install --dev babel-jest @babel/core @babel/preset-env
node index.js
```
